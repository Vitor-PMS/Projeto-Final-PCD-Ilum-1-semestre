# Projeto-Final-PCD-Ilum-1-semestre
Nome: Vitor Pedrique Moura e Silva
Instituição: Ilum Escola de Ciência
Esse é o meu Projeto Final para o curso de PCD no 1º semestre da Ilum. Meu projeto se baseia em uma biblioteca musical, capaz de manipular músicas através de uma interface. Cada música pode receber até 6 atributos: Nome, Artista, Gênero, Ano de Lançamento, Nota, Sei Tocar (o atributo "Sei Tocar" diz respeito a se você sabe ou não tocar uma determinada música). O código é baseado em funções (sendo que algumas funções são suporte para outras funções).
